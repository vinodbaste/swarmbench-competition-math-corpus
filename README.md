# AIME problem-statement mirror (2024–2025)

A frozen mirror of recent AIME problem **statements** only. No solutions and no
answer keys are included.

- `manifest.json` — the index. Its `problems` array lists each problem with an
  `id`, `title`, `text_url` (the statement file under `statements/`), and a
  `source_url` (the original contest page).
- `statements/<id>.md` — one AIME problem statement per file.

Problems are © Mathematical Association of America; this mirror is for
problem-solving practice only.
